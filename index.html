VidFlow Privacy Policy

Last updated: May 2026

What is VidFlow?

VidFlow is a desktop application that helps users automatically schedule and upload videos to YouTube and Instagram using official platform APIs.

What data do we collect?

VidFlow does not collect, store, or transmit any personal data to external servers. All data including your API keys, schedules, and preferences are stored locally on your own computer.

YouTube API Usage

VidFlow uses the YouTube Data API v3 to upload videos on your behalf. We only request the minimum required permission: youtube.upload. We do not read, modify, or delete any existing YouTube data.

Google User Data

VidFlow complies with the Google API Services User Data Policy, including the Limited Use requirements. We do not share your Google data with any third parties.

OAuth Tokens

Authentication tokens received from Google OAuth are stored locally on your device only. They are never uploaded, shared, or transmitted to any server.

Instagram API Usage

VidFlow uses the Instagram Graph API to post Reels on your behalf. Access tokens are stored locally and never shared externally.
